开发一款基于真实物理引擎的 2D 关卡/沙盒游戏。类似于 2D 版的《宇宙沙盒（Universe Sandbox）》或带有解谜性质的引力弹弓游戏。核心功能：多体物理模拟：用户可以在画布上放置不同质量、电荷的天体，模拟引力和库仑力作用下的运动轨迹。游戏模式（引力弹弓解谜）：给定初始天体布局和目标点，玩家需要发射一枚探测器，利用各大质量天体的引力掠过（Gravity Assist）避开障碍物到达终点。时间回溯与轨迹预测：实时绘制未来几秒内的预测轨道（使用数值积分）。数据结构的应用：四叉树（Quadtree）/ 空间哈希（Spatial Hashing）：（重点） 当粒子或星体数量变多时，传统的 $O(N^2)$ 碰撞检测和引力计算会导致卡顿。利用四叉树进行空间分割，可以将碰撞检测和近场力计算优化到 $O(N \log N)$，这是非常硬核且亮眼的数据结构应用。双端队列（Deque）：用于存储天体过去 $N$ 帧的坐标，实现平滑的尾迹（Trail）渲染和“时间倒流”功能。技术栈推荐：Python + Pygame / Arcade 游戏库 + NumPy（用于高效的矩阵与向量物理计算）。
